== Description ==
 
Multi Mobile Redirect allows to set up redirect for any page. You can setup multiple redirects per page.

== Installation ==
 
1. Upload `multi-mobile-redirects` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Under Options in Wordpress menu will appear Mobile Redirect menu
4. Add redirect from page to page
